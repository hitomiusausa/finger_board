## 1. 概要

QuestionBox は旧アプリの「マグネットボックス」と「グループオブジェクト」を統合した問題オブジェクト。学習者がドラッグ操作やテキスト入力で答える問題を作成できる。

**旧アプリとの対応：**

| 旧アプリ | Flutter | 備考 |
|---|---|---|
| マグネットボックス | QuestionBox | ドラッグで答える問題 |
| グループ（自動整列） | QuestionBox（並べ替え型） | 項目の並べ替え問題 |
| 文字入力ボックス | QuestionBox（入力型） | キーボード入力で答える |

---

## 2. QuestionBox の種類

### 2-1. ドラッグ型（Drag）

オブジェクトをドラッグして QuestionBox の中に入れる（または QuestionBox の所定の位置に配置する）ことで解答する。旧アプリの「マグネットボックス」に相当。

**使用例：**
- 絵カードを正しい箱にドラッグして分類する
- 単語を正しい順序に並べる
- パズルのピースを正しい場所に置く

**設定項目：**
- ボックスのサイズ・外観（枠線・背景色）
- アクティブ / 非アクティブ の切り替え
- 編入数制限（最大何個まで入れられるか）
- 自動シャッフル（表示のたびに中の要素をランダム並び替え）
- 複製ドラッグ（元のオブジェクトは残り、コピーがドラッグされる）

### 2-2. 入力型（Text Input）

テキストフィールドに文字を入力して解答する。旧アプリの「文字入力モード付きマグネットボックス」に相当。

**使用例：**
- 空欄に答えを入力する
- 音声を聞いてひらがなで答える

**設定項目：**
- 正解テキスト（複数可）
- ひらがな / カタカナ / 英字 の入力制限
- 音声読み上げ連携

### 2-3. 並べ替え型（Sort）

グループ内のオブジェクトを正しい順序に並べ替えて解答する。旧アプリの「自動整列グループ」に相当。

**使用例：**
- 文章の単語を正しい順に並べる
- 手順を正しい順序に並べる
- 数字を大きい順・小さい順に並べる

**設定項目：**
- 整列方向（横 / 縦）
- 自動シャッフル（表示のたびにランダム順）
- 正解順序の設定

## 3. 正誤判定

### 3-1. ドラッグ型の正誤判定

```
各オブジェクトに「名前」を設定
  ↓
QuestionBox に「正解条件」を設定
  - 「このオブジェクトが入っていれば正解」
  - 「このオブジェクトが入っていなければ正解」
  ↓
学習モードでオブジェクトを配置したとき
  ↓
正誤判定ロジックを実行
  - ✅ 正解 → 正解フィードバック表示 + 進捗記録
  - ❌ 不正解 → 不正解フィードバック表示
```

### 3-2. 入力型の正誤判定

```
入力されたテキストと正解テキストを照合
  - 完全一致 or 部分一致（設定可能）
  - 大文字小文字の区別（設定可能）
  - ひらがな / カタカナの区別（設定可能）
```

### 3-3. 並べ替え型の正誤判定

```
現在の配置順と正解順を照合
  - 全一致で正解
  - 部分正解の表示（設定可能）
```

## 4. 各モードでの QuestionBox の振る舞い

| モード | 振る舞い |
|---|---|
| 編集モード | 設定パネルの表示・正解条件の設定・アクティブ/非アクティブ切替 |
| 提示モード | 答えを表示する（教師が見せる側）。正誤判定は発動しない |
| 学習モード | 学習者が解答する。正誤判定が発動し、進捗が記録される |

---

## 5. フィードバック表示

### 5-1. 正解時

- ✅ アイコン + 「正解！」テキスト
- 効果音（設定可能）
- アニメーション（パーティクル等）

### 5-2. 不正解時

- ❌ アイコン + 「もう一度」テキスト
- 効果音（設定可能）
- オブジェクトを元の位置に戻す

### 5-3. フィードバックのカスタマイズ（将来対応）

- フィードバックの有無
- 不正解時にヒントを表示
- 試行回数の制限

## 6. progress テーブルへの記録

QuestionBox への解答は `progress` テーブルに記録する。

```sql
INSERT INTO progress (
  student_id,   -- 解答した生徒の id
  material_id,  -- 教材の id
  page_id,      -- ページの id
  object_id,    -- QuestionBox の BoardObject.id ← Dashboard の集計で使う
  is_correct,   -- 正解 / 不正解
  score,        -- スコア（0 or 1、将来的には部分点対応）
  answer_given, -- 実際に入力・配置された内容（JSON文字列）
  answered_at,  -- 解答日時
  session_id    -- 学習セッションの id
)
```

**重要：** `object_id` は `BoardObject.id` と一致させること。Phase 5 の Dashboard での集計に使用する。

## 7. ブラインドモード（Phase 2 後半）

教師が事前に「隠す」設定をしたオブジェクトを、学習モード開始時に非表示にする機能。

**用途：** 答えの絵カードを隠した状態で問題を提示し、正解すると表示される、等。

**実装方針：** BoardObject に `isBlind: bool` プロパティを追加し、学習モードで `isBlind=true` のオブジェクトを非表示にする。
